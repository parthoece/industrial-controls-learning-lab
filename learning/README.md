# Learning materials

The lessons are self-contained at fundamentals level.

External references are optional for the first pass. Use them for standards detail, vendor behavior, and advanced study.

## How to study each week

1. Read the lesson.
2. Redraw the main diagram.
3. Answer the knowledge checks.
4. Run the linked lab.
5. Change one assumption.
6. Test a failure.
7. Write an experiment report.
8. Read one deep-study source.

## Definition of concept-ready

You are concept-ready when you can:

- explain the idea without vendor jargon
- name the important units, states, and timing assumptions
- trace normal behavior
- predict one boundary case
- diagnose one failure case
- connect the lesson to code or a data model
- state what the simplified model does not prove

## Reference rules

- Core lesson first.
- Official or primary source second.
- Vendor documentation for platform detail.
- Never copy standards or vendor text into notes.
- Record access date and uncertainty.
