# Reference Policy and Starter Sources

Prefer current official documentation, standards bodies, textbooks, and primary technical papers. Record the access date because product documentation changes.

## Controls and PLC programming

- Beckhoff InfoSys: TwinCAT 3 PLC introduction
- PLCopen Motion Control overview
- official IEC 61131-3 vendor language documentation
- a university or textbook source for introductory feedback control

## Motion and industrial networks

- EtherCAT Technology Group technology and diagnostic material
- CAN in Automation CiA 402 overview
- official drive-vendor controlword/statusword documentation, labeled vendor-specific
- current motion-library documentation for any function block discussed

## Smart manufacturing

- OPC Foundation OPC UA material
- OASIS MQTT specification
- ISA-95 overview and official supporting material
- official database and API framework documentation

## Open-source maintenance

- GitHub documentation for community health files and issue forms
- Contributor Covenant
- Apache License 2.0

## Note quality rule

Do not contribute a bare link list. Every reference note should state:

1. the problem the source helps solve
2. the relevant layer in the system
3. the learner's interpretation
4. an example or experiment
5. what still needs verification
